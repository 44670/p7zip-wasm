t (Test integrity of archive) command

Tests archive files.

Example

7z t archive.zip *.doc -r

tests *.doc files in archive archive.zip.

Switches that can be used with this command

-ai (Include archives)
-an (Disable parsing of archive_name)
-ai (Exclude archives)
-i (Include)
-p (Set Password)
-r (Recurse)
-x (Exclude)
